# ROADMAP.md

**Last Updated:** May 28, 2026

## Vision

Project Genesis is evolving into a scientific creature stewardship and genetics management simulation.

The player breeds, studies, manages, and preserves life inside Sanctuary Site-04 through predictive genetics, lineage tracking, habitat management, and long-term sanctuary operations.

## Phase 1: Foundation (Completed)

- Core data-driven creature system
- Foundation creatures loaded from JSON
- Trait library
- CreatureManager
- BreedingManager / BreedingEngine
- SimulationAdapter
- SanctuarySite04 hub shell
- BreedingLabUI integration
- public/private documentation workflow

## Phase 2: Readability & Management Foundation (Current Priority)

Current emphasis:
- make creatures readable
- make breeding strategic
- make registry management scalable

Priority order:
1. CreatureCard.tscn
2. Registry UX overhaul
3. Parent slot visual upgrade
4. Trait color coding
5. Offspring prediction range/risk display
6. Lineage viewer
7. Habitat Dome

## Phase 3: Persistence & Simulation Depth

- Save/load system
- persistent generated creatures
- expanded event logging
- lineage persistence
- habitat compatibility scoring
- environmental fitness
- simulation ticking

## Phase 4: Advanced Genetics

- deeper D/R/M dominance conflicts
- recessive trait surfacing
- genome purity
- instability risk
- inherited defects
- hybridization risk
- Genome Sequencer facility

## Phase 5: Sanctuary Expansion

- multiple habitats
- observatory/archive systems
- research systems
- contamination mechanics
- narrative event chains
- expedition systems

## Long-Term Vision

A living sanctuary where each generation matters, every creature has lineage, and biological decisions carry visible strategic and emotional consequences.
